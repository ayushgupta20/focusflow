## ✨ Features

- **🍅 Advanced Timer Logic:** Deep Work (50/10), Pomodoro (25/5), Light (15/3), or Custom modes.
- **🎧 Ambient Soundscapes:** Built locally via the Web Audio API—no internet required. Focus with Rain, Ocean, Forest, or White Noise.
- **🎮 Gamification:** Level up as you focus! Earn XP for completing tasks and focus sessions, and collect up to 12 unique achievement badges.
- **🚀 Multi-User Accounts:** Securely register, login, and manage individual profiles right in the browser (using local machine storage).
- **📋 Smart Task List:** Manage your to-dos with priority tags (High/Med/Low) and map active tasks directly to your timer.
- **🎨 Glassmorphism & Themes:** Sleek dark mode design with real-time particle backgrounds. Switch between Violet, Ocean, Sunset, and Forest themes on the fly.
- **🧘 Breathing Guide:** A built-in animated breathing guide to help you calm your mind before deep focus.

## 💻 Tech Stack

- **HTML5** (Semantic structuring)
- **Vanilla CSS** (CSS Variables, Flexbox/Grid, Animations)
- **Vanilla JavaScript** (ES6+, DOM Manipulation, LocalStorage)
- **Web Audio API** (Synthesized ambient noise)

## 🌐 Live Demo & Hosting

This app is designed as a **Static Site** and requires absolutely zero backend.

**To host this for free on the WWW and share it on any device:**
1. Navigate to the **Settings** tab of this GitHub Repository.
2. Click on **Pages** in the left sidebar.
3. Under **Build and deployment**, select **Deploy from a branch**.
4. Set the branch to **`main`** and the folder to `/ (root)`.
5. Click **Save**.

Within 1-2 minutes, GitHub will provide you with a live URL (e.g., `https://ayushgupta20.github.io/focusflow/`) that works on any smartphone, tablet, or desktop in the world!

## 🚀 Local Development

Since FocusFlow uses raw HTML/CSS/JS, you can simply clone it and open it in your browser:

```bash
# Clone the repository
git clone https://github.com/ayushgupta20/focusflow.git

# Navigate into the directory
cd focusflow

# Serve the application locally (requires Node.js)
npx serve .
```

Navigate to `http://localhost:3000` to start focusing.

## 🔒 Privacy

Your personal data is yours. FocusFlow tracks your sessions, XP, and profile details entirely inside your device's browser `localStorage`. We do not track you or send your data to external databases.
